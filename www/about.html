About The Project

How Does DocuLink Work

 DocuLink maintains a searchable database of websites' urls, titles, and descriptions. All searchable entries will be provided by you, the user.

Goals

    DocuLink strives to be a simple, no bloat, community driven search engine. All searchable entries will be supplied by the users, without any sort of validation process. DocuLink uses POST requests rather than GET requests to provide an extra layer of privacy for the user. Ideally DocuLink should be kept as light as possible, meaning favouring CSS for animations, and styling over JavaScript. DocuLink should be 100% functional with the LibreJS browser extension, all JavaScript code will have a magnet link to it's license commented above it's source code.

Rules For This Project

  • All JavaScript must be compatible with LibreJS extension
  • All HTML should comply with the W3C's Living Standard
